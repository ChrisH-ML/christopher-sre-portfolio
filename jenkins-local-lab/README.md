\# Jenkins Local CI/CD Lab



This project documents a local Jenkins environment created to learn CI/CD pipeline concepts.



\## Setup



Jenkins was installed locally and accessed at:



http://localhost:8080



\## Pipeline Example



The pipeline demonstrates a simple CI workflow with build, test, and deploy stages.



\## Purpose



This lab was created to understand:



\- CI/CD pipeline structure

\- Jenkins pipeline syntax

\- automation of build workflows
